Experiment 01

2026-06-13 · single prompt · 1 baseline · 4 challengers · 3 controls

The prompt

Prompt: Create a webbed Flappy Bird video game, but 3D and I want it to look pretty and playable.

Contenders

Each contender lives in its own MODEL_NAME/ folder containing the result HTML, a SPEC.md (params, for comparing) and a JOURNEY.md (thinking process). Markdown opens in the in-browser viewer.

  • Baseline · fable-5 — claude-fable-5, no Fabled skill. Delivered (36/36 assertions, ~$2.30 est.).
    Result → · SPEC.md · JOURNEY.md
  • Challenger · variant glm-5-1-fabled — glm-5.1 + Fabled skill on Pi. Delivered (72,086 assertions, ~$0.73).
    Result → · SPEC.md · JOURNEY.md
  • Challenger · variant gpt-5-5-codex-fabled — gpt-5.5 + Fabled skill on Codex. Delivered (browser-verified, 0 page errors, 0 console errors).
    Result → · SPEC.md · JOURNEY.md

Result

Both runnable builds shipped a complete single-file 3D Flappy Bird with a pure logic/render split and passing assertions. The Fabled challenger on Pi delivered for ~$0.73 against the baseline's estimated ~$2.30, with far more assertions (72,086 vs 36) — though the baseline's cost bundles plan-derivation the skill provides for free, so read it as indicative, not a clean ledger. The same GLM-5.1 + Fabled combination on OpenCode failed: the stream closed roughly every 60 s in a loop and produced no deliverable — full analysis in its REPORT.md. The control (glm-5-1-pi) ran the same model/agent with no Fabled skill: it shipped a single polished file verified directly in a real browser (0 console errors) for ~$0.50 in ~7.6 min, but skipped the logic/render split and 72k-assertion test suite the skill drove — making the delta the value of the skill itself. The Codex challenger (gpt-5-5-codex-fabled) shipped the same prompt as a single-file Three.js game, verified through local Chrome with no page or console errors; it used 196,410 total tokens plus 1,146,624 cached tokens for $2.874655 exact ($2.87 rounded). Exact return time was not recorded; the checked lower bound is 33m56s. The Codex control (gpt-5-5-codex) shipped a browser-verified Three.js build with score reaching 3, 3,019,341 total tokens, 1,243.055 s wall time, and $5.27709 calculated cost. The gpt-5-mini + Fabled challenger (gpt-5-mini-gh-pi-fabled) is the cheapest delivered run in exp-01 — 78,402 tokens, $0.0339, ~2.1 min active — and followed the full Fabled process (contracts, no placeholders, evidence-backed checklist). Caveats: turn 1 emitted the code as chat text without writing the file (fixed by a nudge), and Phase 4 ran as a static trace (Mode A), so the artifact was not executed in a browser.
