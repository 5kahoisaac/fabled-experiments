Who builds the whole thing?

A running log of single-prompt builds: Fable 5 as the baseline, a cheaper model wearing the Fabled skill as the challenger, and another model without Fabled as the control — each given a single prompt and asked to ship a complete, runnable project. Each run measures whether a cheaper model plus the right skill can match the flagship — and what it costs.

Experiments

Each card compares several single-prompt builds of the same prompt — a baseline, challengers, and a control.

How a run works

  1. 01

    Same prompt, sealed

    Both contenders get an identical single prompt — no follow-ups, no nudges.

  2. 02

    Three roles, one prompt

    The baseline is Fable 5. A challenger runs the Fabled skill on a cheaper model (it may succeed or fail). A control runs another model without Fabled, as reference.

  3. 03

    Measure the cost of completeness

    Tokens, dollars, wall-clock, and whether the output actually runs end-to-end.
